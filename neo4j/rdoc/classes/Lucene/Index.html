Class Lucene::Index
In: lib/lucene/index.rb
Parent: Object

Represents a Lucene Index. The index is written/updated only when the commit method is called. This is done since writing to the index file should be done as a batch operation. (Performace will be bad otherwise).

new(path)

Returns an Index instance for the current running transaction.

Tries to reuse an Index instance for the current running transaction. If a Lucene::Transaction is running it will register this index in that transaction if this has not already been done. When it has been registered in the transaction the transaction will commit the index when the transaction is finished. The configuration (kept in the field_infos) for this index will be the same for all indexes with the same path/key.

Parameters

path<String>:The key or location where the index should be stored (relative Lucene::Config[:storage_path]

Examples

Index.new ‘foo/lucene-db‘

Returns

Returns a new or an already existing Index

commit()

Writes to the index files Open and closes an lucene IndexWriter Close the IndexSearcher so that it will read the updated index next time. This method will automatically be called from a Lucene::Transaction if it was running when the index was created.

[Source]

168:     def commit
169:       # TODO not enough to block threads here, since the @uncommited and @deleted_ids may change
170:       # Need to synchronize 
171:       lock.synchronize do
172:         $LUCENE_LOGGER.debug "  BEGIN: COMMIT"
173:         delete_documents # deletes all docuements given @deleted_ids
174:       
175:         # are any updated document deleted ?
176:         deleted_ids = @uncommited.keys & @deleted_ids
177:         # delete them those
178:         deleted_ids.each {|id| @uncommited.delete(id)}
179:       
180:         # update the remaining documents that has not been deleted
181:         update_documents # update @documents
182:         
183:         @uncommited.clear  # TODO: should we do this in an ensure block ?
184:         @deleted_ids.clear
185:         
186:         # if we are running in a transaction remove this so it will not be commited twice
187:         Transaction.current.deregister_index(self) if Transaction.running?
188:         $LUCENE_LOGGER.debug "  END: COMMIT"        
189:       end
190:     rescue => ex
191:       $LUCENE_LOGGER.error(ex)
192:       #      ex.backtrace.each{|x| $LUCENE_LOGGER.error(x)}
193:       raise ex
194:     end
