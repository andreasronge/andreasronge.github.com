Creates a relationship between this and the other node. Returns self so that we can add several nodes like this: 

  n1 = Node.new # Node has declared having a friend type of relationship
  n2 = Node.new
  n3 = NodeMixin.new

  n1 << n2 << n3

This is the same as: 

  n1.friends.new(n2)
  n1.friends.new(n3)
