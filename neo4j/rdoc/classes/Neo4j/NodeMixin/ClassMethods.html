create_value_class()

The struct will have the Ruby on Rails method: model_name and new_record? so that it can be used for restful routing.

find(query=nil, &block)

Finds all nodes of this type (and ancestors of this type) having the specified property values.

Example

  MyNode.find(:name => 'foo', :company => 'bar')

Or using a DSL query 

  MyNode.find{(name == 'foo') & (company == 'bar')}

See the lucene module for more information how to do a query.

index(*rel_type_props)

Index a property a relationship. If the rel_prop arg contains a ’.’ then it will index the relationship. For example "friends.name" will index each node with property name in the relationship friends. For example "name" will index the name property of this NodeMixin class.

423:         clazz = relations_info[rel_type.to_sym][:class]
424:         
425:         type = relations_info[rel_type.to_sym][:type]  # this or the other node we index ?
426:         rel_type = type.to_sym unless type.nil?
427:         
428:         # updater - called when index needs to be updated
429:         updater = lambda do |my_node, doc|

432:           relations.each {|other_node| values << other_node.send(prop)}
433:           doc[index_key] = values

435:         index_updaters[index_key] = updater
436:       
437:         # trigger - knows if an index needs to be updated
438:         trigger = lambda do |other_node, event|
439:           if (Neo4j::PropertyChangedEvent.trigger?(event, :property, prop) or

441:                 Neo4j::NodeLifecycleEvent.trigger?(event))
442:             relations = other_node.relations.both(rel_type).nodes
