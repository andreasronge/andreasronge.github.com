start

public start

Starts neo unless it is not already started. Before using neo it has to be started and the location of the Neo database on the filesystem must have been configured, Neo4j::Config[:storage_path].

Examples

Neo4j::Config[:storage_path] = ’/var/neo4j-db’ Neo4j.start

def self.start
  return if @instance
  @instance = Neo.new 
  @instance.start
  at_exit do
    Neo4j.stop
  end
  @instance
end
